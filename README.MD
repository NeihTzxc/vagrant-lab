# Vagrant Commands

## `vagrant box add`

Downloads and installs a new box from a specified URL.

Example: `vagrant box add ubuntu/focal64`

## `vagrant box list`

Lists all available boxes on the current system.

Example: `vagrant box list`

## `vagrant box remove`

Deletes an installed box.

Example: `vagrant box remove ubuntu/focal64`

## `vagrant cloud`

Interacts with the Vagrant Cloud service.

Example: `vagrant cloud auth login`

## `vagrant destroy`

Deletes the virtual machine and all associated resources.

Example: `vagrant destroy`

## `vagrant global-status`

Displays a list of all running Vagrant environments on the current system.

Example: `vagrant global-status`

## `vagrant halt`

Stops the virtual machine.

Example: `vagrant halt`

## `vagrant help`

Displays help information for Vagrant.

Example: `vagrant help`

## `vagrant init`

Creates a new Vagrantfile in the current directory, using a specified box as the starting point.

Example: `vagrant init ubuntu/focal64`

## `vagrant login`

Logs in to the Vagrant Cloud service.

Example: `vagrant login`

## `vagrant package`

Packages the virtual machine as a reusable box file.

Example: `vagrant package --output ubuntu-focal64.box`

## `vagrant plugin`

Manages Vagrant plugins.

Example: `vagrant plugin install vagrant-aws`

## `vagrant port`

Displays the port mappings between the host machine and the virtual machine.

Example: `vagrant port`

## `vagrant provision`

Runs the provisioning scripts defined in the Vagrantfile.

Example: `vagrant provision`

## `vagrant reload`

Restarts the virtual machine, reloading any updated configuration settings.

Example: `vagrant reload`

## `vagrant resume`

Resumes a previously halted virtual machine.

Example: `vagrant resume`

## `vagrant share`

Shares a Vagrant environment with others.

Example: `vagrant share`

## `vagrant ssh`

Connects to the virtual machine via SSH.

Example: `vagrant ssh`

## `vagrant ssh-config`

Displays the SSH configuration for connecting to the virtual machine.

Example: `vagrant ssh-config`

## `vagrant status`

Displays the status of the virtual machine (running, stopped, etc.).

Example: `vagrant status`

## `vagrant suspend`

Suspends the virtual machine.

Example: `vagrant suspend`

## `vagrant up`

Launches and provisions a new virtual machine based on the settings in the Vagrantfile.

Example: `vagrant up`

## `vagrant version`

Displays the version of Vagrant.

Example: `vagrant version`

## `vagrant snapshot list`

Lists all available snapshots for the virtual machine.

Example: `vagrant snapshot list`

## `vagrant snapshot save`

Creates a new snapshot of the virtual machine.

Example: `vagrant snapshot save my_snapshot`

## `vagrant snapshot restore`

Restores a previously saved snapshot of the virtual machine.

Example: `vagrant snapshot restore my_snapshot`

These are all the available Vagrant commands with examples for each command. For more detailed information on each command and its options, you can run `vagrant help <command>` or visit the Vagrant documentation at https://www.vagrantup.com/docs/cli/.
